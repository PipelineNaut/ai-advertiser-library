Lesson 5.3 — Module 5: AI Integration

Connecting Software to Meta API

Step-by-Step ~20 min Needs: Meta Business Manager (Module 3)
We're connecting the software to your Facebook ad account. Kind of tedious but you only do it once. ~20 minutes.

🎯 What Is an API?

An API is basically a secret tunnel between the software and Facebook. Instead of you clicking buttons on Facebook to create ads, the software uses this tunnel to do it for you.

You set it up once. After that the software can create ads, check results, turn stuff off — without you touching Facebook.

📋 Before You Start — Checklist

Connection Progress 0 / 0 complete

⚠️ If you're managing a client's account, THEY need to give you access through their Business Manager. Don't skip this. If you're missing anything, pause and go back to Module 3.

🌐 Step 1: Go to Meta Developers

  1. Open your browser → go to developers.facebook.com
  2. Click Log In (top right) — use the same Facebook account linked to your Business Manager
  3. You'll land on the Meta for Developers dashboard
  4. Click My Apps (top right)
  5. Click Create App (green button)

⚙️ Step 2: Create Your App

  1. App type: select Business
  2. App name: something like My Agency Ads Tool (doesn't matter, only you see this)
  3. Contact email: your email
  4. Business portfolio: select your Business Manager
  5. Click Create App
  6. You might get a security check (captcha) — just complete it

"App" sounds scary — it's not. It's just a container that holds your connection keys. You're not building anything.

🔑 Step 3: Get Your Access Token

  1. In your app dashboard, go to Tools in the left menu
  2. Click Graph API Explorer
  3. On the right side, you'll see a dropdown for User or Page — select User Token
  4. Click Add a Permission → search and add these one by one:
ads_management
Lets the software create and edit ads
ads_read
Lets the software see your ad results
business_management
Lets it access your Business Manager
pages_read_engagement
Lets it read your Page info
  1. Click Generate Access Token (blue button)
  2. A popup asks you to authorize — click Continue through it
  3. Copy that long token (the string of random letters and numbers)

🔴 NEVER share this token with anyone. Treat it like your bank password. Anyone with this token can control your ad account.

Heads up: this token expires after ~1 hour. The software will walk you through getting a long-lived one (lasts 60 days).

📋 Step 4: Paste Into the Software

  1. Open the software
  2. Go to Settings → Meta Connection (or API Settings)
  3. Paste your access token
  4. Enter your Ad Account ID — find this in Business Manager → Ad Accounts → the number that starts with act_
  5. Click Connect or Save

🧪 Step 5: Test the Connection

  1. In the software, go to Campaigns or Dashboard
  2. If the connection works, you'll see your existing campaigns show up
  3. If you don't have any campaigns yet, you'll see an empty list — that's fine, it means it connected

If you see your campaigns (or an empty list with no errors) — you're good. Connection works.

🔧 ⚠️ Troubleshooting

1. "Invalid token"

Token expired. Go back to Graph API Explorer, generate a new one.

2. "No ad account found"

Double-check your Ad Account ID. Make sure it starts with act_.

3. "Permissions error"

Go back and make sure you added ALL 4 permissions listed in Step 3.

4. "Business not found"

You need to be an admin on the Business Manager, not just an employee.
