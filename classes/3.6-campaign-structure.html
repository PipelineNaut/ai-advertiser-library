Campaign Structure — How Ads Are Organized

Campaign Strategy Beginner ~12 min
Before you create your first ad, you need to understand how Meta organizes everything. It's three levels: Campaign → Ad Set → Ad. Get this wrong and your ads underperform from day one. Get it right and everything else becomes easier.

🖥️ Screens to Open (Have These Ready)

1. Ads Manager — Campaigns Tab — Show the 3-level structure (campaigns → ad sets → ads) 2. Campaign Creation Flow — Show objective selection (pick "Leads"), CBO toggle, naming 3. Ad Set Level — Show audience targeting, location radius, age, daily budget field 4. Ad Level — Show creative upload, primary text, headline, CTA button dropdown

🏗️ 1. The Three Levels

🍕 The Pizza Shop Version

Think of it like running Tony's restaurant:

Campaign = the overall goal. "I want more customers this month."
Ad Set = WHO you're targeting and HOW MUCH you're spending. "Families within 5 miles, $15/day."
Ad = what people actually see. "A photo of a pepperoni pizza with the headline 'Family Deal: 2 Large Pizzas $19.99'."

One campaign can have multiple ad sets (different audiences). Each ad set can have multiple ads (different creatives). Meta tests them all and pushes budget toward what works best.

🖥️ SHOW ON SCREEN: Open Ads Manager → Campaigns tab. Click into any campaign to show the 3 levels: Campaigns → Ad Sets → Ads tabs at the top. Point to each tab as you explain.
📁 Campaign
The Goal
What do you want? Leads? Traffic? Sales? You pick ONE objective per campaign. This tells Meta's AI what to optimize for. For local businesses, you'll almost always pick "Leads".
📂 Ad Set
The Targeting + Budget
WHO sees the ad (location, age, interests) and HOW MUCH you spend. Think of each ad set as a different "audience experiment." You might test: Ad Set 1 = pizza lovers within 5 miles, Ad Set 2 = families within 10 miles.
📄 Ad
The Creative
WHAT people see — the image or video, the text, the headline, the call-to-action button. This is the only part your client's customers interact with. Everything else is behind the scenes.

💡 The hierarchy matters. You can't create an ad without an ad set. You can't create an ad set without a campaign. Always build top-down: Campaign first → Ad Set → Ad.

💰 2. Where Does the Budget Go? (CBO vs ABO)

Budget can live at the campaign level or the ad set level. This is one of the most confusing things for beginners, but it's actually simple:

🖥️ SHOW ON SCREEN: Click "+ Create" → pick "Leads" objective → scroll to "Campaign Budget Optimization" toggle. Show it ON (CBO) and OFF (ABO). Point out where the daily budget field moves depending on which is selected.

CBO — Campaign Budget Optimization

You set ONE budget at the campaign level. Meta decides how to split it across your ad sets.

  • How it works: You say "$30/day." Meta gives $20 to the ad set that's performing best and $10 to the other.
  • Good for: When you have proven audiences and want Meta to optimize automatically.
  • Bad for: Testing, because Meta might starve a new ad set before giving it a fair chance.

ABO — Ad Set Budget Optimization

You set a budget for EACH ad set individually. You're in full control.

  • How it works: You say "$15/day for Ad Set 1" and "$15/day for Ad Set 2." Each gets exactly what you assigned.
  • Good for: Testing new audiences and creatives. Each test gets equal spend.
  • Bad for: Scaling, because you're manually managing every ad set's budget.
The Simple Rule
🧪 Testing?
Use ABO
→
🚀 Scaling?
Use CBO
Start every client on ABO. Move winning ad sets into a CBO campaign once you know what works.

🏷️ 3. Naming Your Campaigns (Stay Organized)

When you have 5 clients with 3 campaigns each, things get messy fast. Use a consistent naming system from day one:

🖥️ SHOW ON SCREEN: Show your Ads Manager with a real or example campaign. Point to how you've named the campaign, ad sets, and ads. Show how easy it is to find things when they're named properly vs "Ad Set 1, Ad Set 2, Ad Set 3."
Campaign: [Client] - [Goal] - [Type]
Example: Tony's Pizza - Leads - ABO Testing
Example: Tony's Pizza - Leads - CBO Scale
Ad Set: [Audience] - [Location] - [Age]
Example: Pizza Lovers - Tampa 5mi - 25-55
Example: Families - Tampa 10mi - 30-50
Ad: [Format] - [Hook/Angle] - [Version]
Example: Image - Family Deal $19.99 - V1
Example: Video - Behind the Kitchen - V2

💡 Future you will thank present you. At 2am when a client texts "why did my ad stop?" you need to find the right campaign in 5 seconds. Good naming makes that possible.

🧪 5. The Testing Framework

Running ads isn't "set it and forget it." It's a process of testing, learning, and optimizing. Here's the framework:

🖥️ SHOW ON SCREEN: Open an active campaign with multiple ads. Show the "Delivery" column (Active, Learning, Learning Limited). Show the "Cost per result" column. Point to which ads are winning and which you'd kill. Show how to turn an ad off (toggle switch).
1
Launch 3-5 different ads to the same audience
Different images/videos, different headlines, different angles. Keep the audience the same so you're only testing the creative.
2
Wait for data — don't panic
Let each ad spend at least 2-3x your target cost per lead before making any decisions. If your target CPL is $15, wait until each ad has spent $30-$45. This usually takes 3-5 days.
3
Kill the losers
If an ad has spent 3x your target CPL and still has zero or very few leads, turn it off. Don't hope it gets better. It won't. Redirect that budget to what's working.
4
Scale the winners slowly
Found an ad that's crushing it? Increase budget by 20-30% every 2-3 days. Going from $15/day to $20/day is fine. Going from $15/day to $50/day overnight will reset the learning phase and tank performance.
5
Repeat forever
Even winners fatigue eventually. When CPL starts rising, launch new creatives. The cycle never stops: test → kill losers → scale winners → test new creatives.

🔴 The #1 beginner mistake: changing things too fast. You launch an ad, see no leads after 24 hours, and panic-edit the targeting. Meta needs 3-5 days to learn who to show your ad to (this is called the "learning phase"). Every time you edit the ad set, the learning phase resets. Be patient.

🧠 6. The Learning Phase (Don't Touch It)

When you launch a new ad set, Meta enters what's called the "learning phase." During this phase, Meta is figuring out who to show your ad to. Performance will be inconsistent — some days great, some days terrible. This is normal.

🖥️ SHOW ON SCREEN: Find an ad set that says "Learning" or "Learning Limited" in the Delivery column. Show where it says this. If you have one that's exited learning ("Active"), show the difference in performance stability.
⏱️
How long?
Usually 3-7 days or ~50 conversion events, whichever comes first
🚫
What resets it?
Editing targeting, changing budget by more than 20%, pausing for more than a few hours
✅
What's safe to edit?
Ad creative (new ads), bid caps, and small budget increases (under 20%)

⚠️ Tell your client upfront: "The first week might look rough. That's normal — Meta is learning. Don't freak out. By week 2, we'll have real data to work with." Set this expectation on the onboarding call so they don't panic-text you on day 2.

📋 7. Your First Campaign Structure (Copy This)

Here's the exact structure for your first client's campaign. Don't overthink it:

🖥️ SHOW ON SCREEN: Build this structure live. Click "+ Create" → Leads → name it → create the first ad set with broad targeting → set location radius to 10 miles around a city → set age 25-55 → set $15/day budget. Walk through each field as you fill it in. This is the "follow along with me" moment.
📁 Campaign: [Client Name] - Leads - ABO Testing
Objective: Leads | Budget: ABO (ad set level)
📂 Ad Set 1: Broad - [City] 10mi - 25-55
$10-15/day | Broad targeting (let Meta's AI find the right people)
📄 Ad 1: Image - [Offer Angle A] - V1
📄 Ad 2: Image - [Offer Angle B] - V1
📄 Ad 3: Video - [Testimonial/BTS] - V1
📂 Ad Set 2: Interest - [Relevant Interest] - 25-55
$10-15/day | Interest-based targeting (e.g., "Pizza", "Italian food", "DoorDash")
📄 Ad 1: Same 3 ads as above
(Same creatives, different audience — this way you're testing audiences, not creatives)

💡 Total daily spend: $20-30/day. That's $600-900/month. Enough to generate real data and real leads. Don't let clients start below $10/day — Meta doesn't have enough budget to learn at that level.
