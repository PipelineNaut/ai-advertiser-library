Lesson 5.2 — Module 5: AI Integration

Claude Code Setup (Step-by-Step)

Hands-On ~15 min You need: computer + internet
Open your laptop and follow along. We're installing Claude Code — takes about 15 minutes. You don't need to know how to code.

🎯 What Is Claude Code?

Claude Code is an AI you can talk to in plain English. You ask it stuff, it writes ad copy, does research, analyzes data. Made by Anthropic (same world as ChatGPT, different company).

It lives inside VS Code — a free app from Microsoft. Don't let the name scare you, we're literally only using it as a chat window to talk to Claude.

💻 Step 1: Download VS Code

  1. Open your browser → go to code.visualstudio.com
  2. Click the big blue download button (it picks your computer type automatically)
  3. Mac: Open the downloaded file → drag it to Applications
  4. Windows: Open the downloaded file → click Install
  5. Open VS Code — you'll see a dark screen with a welcome tab. That's normal.

⚠️ Windows users — one extra step: Also download Git Bash from gitforwindows.org. Claude Code needs it to talk to your computer properly. Just install it, takes 2 minutes.

🔌 Step 2: Install the Claude Code Extension

  1. In VS Code, look at the left sidebar — see the icon that looks like 4 squares (like building blocks)? Click it.
  2. That opens the Extensions panel (extensions = add-ons, like apps for your phone)
  3. In the search bar at the top, type "Claude Code"
  4. Click the one by Anthropic (look for the verified checkmark ✓)
  5. Click the blue Install button
  6. Wait 10 seconds. Done.

🔐 Step 3: Sign In

  1. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac) — this opens the command search bar
  2. Type "Claude: Sign In" and click it
  3. Your browser opens → create an Anthropic account (or sign in if you have one)
  4. Pick a plan — the Pro plan ($20/mo) is what you want
  5. Go back to VS Code — you should see a Claude chat panel

💡 If the chat panel doesn't appear: click View in the top menu → Claude Code

📁 Step 4: Create Your HQ

This folder is where your AI agent lives. Everything — your client files, ad copy, reports — goes here. Pick a name you actually like, because you'll open this folder every day.

  1. In VS Code, click File → Open Folder

📄 Step 6: Create Your CLAUDE.md File

This file is your agent's brain. It reads it every time you start a conversation — who it is, how it should talk, what it knows about your business. Pick the template that matches the personality you chose above.

  1. Right-click in the left file panel → New File
  2. Name it exactly: CLAUDE.md
  3. Copy-paste the template for your chosen agent below
  4. Fill in the [brackets] with your info and save

🔧 ⚠️ Troubleshooting — 3 Most Common Issues

1. "Claude not responding"

Close VS Code, reopen it. 90% of problems = restart.

2. "Extension not found"

Make sure you're searching in the Extensions panel (4-squares icon), not the regular search bar.

3. "Sign-in not working"

Clear your browser cookies for anthropic.com, try again.
